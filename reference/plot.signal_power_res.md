# Plot method for signal_power_res objects

Displays the cumulative power spectrum for each evaluated signal and
marks the frequency cutoff used to derive the recommended downsampling
factor.

## Usage

``` r
# S3 method for class 'signal_power_res'
plot(x, ...)
```

## Arguments

- x:

  An object of class \`"signal_power_res"\`.

- ...:

  Additional arguments (not used).

## Value

A \`ggplot\` object, returned invisibly.
